Xbox Live Clan CMS
================
PHP basiertes Content Management System für XBox Live Clans
--
___
### System-Anforderungen ###
- PHP 5.3+
	- MCrypt
- MySQL
- xboxapi.com-API-Key
        
### System-Umfang ###
- WebApp-Ansicht (Für Mobile Endgeräte)
- Signaturen Generator (inkl. PSD)
- JoinUs
- News
- Forum
- Mitglieder/Gruppen-Übersicht
